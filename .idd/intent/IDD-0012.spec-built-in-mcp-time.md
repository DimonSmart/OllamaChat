# IDD-0012.spec-built-in-mcp-time

## Intent

Provide an opt-in built-in MCP server for obtaining current time information.

## Related Specifications

Related: IDD-0007, IDD-0008.

## Behavior

The server returns current time in ISO-8601 form and accepts an optional IANA or Windows time-zone identifier.

## Durable Architecture And Constraints

The tool is read-only.

## Non-Goals

The server does not persist schedules or modify system time.

## Acceptance Criteria

- Omitted time zone has deterministic default handling.
- Supported IANA and Windows identifiers resolve to the requested zone.

## Verification

Automated coverage verifies time-zone resolution and structured result shape.
