# IDD-0007.spec-mcp-server-management

## Intent

Let users discover and manage external and built-in MCP server connections available to agents.

## Related Specifications

Related: IDD-0005, IDD-0008, IDD-0011, IDD-0012, IDD-0013, IDD-0014, IDD-0015, IDD-0016, IDD-0017, IDD-0018, IDD-0019.

## Behavior

Users can search configured servers; create, edit, and remove local-command or remote-SSE connections; import a compatible MCP installation link; and configure built-in server settings through their dedicated editor.

## Durable Architecture And Constraints

Tool connections are distinct from agent templates so they can be bound selectively to agents.

## Non-Goals

This surface does not invoke a tool; interactive invocation belongs to the MCP playground.

## Acceptance Criteria

- Users can distinguish built-in from configured external servers.
- Local and remote connection modes collect their relevant connection information.
- A parsed installation link produces an editable server configuration.

## Verification

Automated coverage verifies configuration validation, link parsing, binding selection, and built-in server protection.
