# IDD-0004.spec-vector-search

## Intent

Enable retrieval over files attached to an agent and show index progress and search results.

## Related Specifications

Related: IDD-0001, IDD-0005, IDD-0010.

## Behavior

Users select an agent, see its attached-file and indexing state, submit a query, and receive ranked content segments with file identity and score. The same agent-bound knowledge can supply context during chat.

## Durable Architecture And Constraints

Retrieval is scoped to the selected agent's attached files.

## Non-Goals

This surface does not manage general-purpose document libraries.

## Acceptance Criteria

- Search results identify their source file and relevance score.
- Indexing state remains visible while processing is in progress.
- Chat retrieval does not use files from unrelated agents.

## Verification

Automated coverage verifies indexing recovery, agent isolation, and ranked retrieval behavior.
