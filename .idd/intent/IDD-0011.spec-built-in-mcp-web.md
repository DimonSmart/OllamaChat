# IDD-0011.spec-built-in-mcp-web

## Intent

Provide an opt-in built-in MCP server that searches the web and retrieves page content for agent use.

## Related Specifications

Related: IDD-0007, IDD-0008.

## Behavior

The server accepts a search query and bounded result count, returns structured search results, and can retrieve source content from a selected result or URL.

## Durable Architecture And Constraints

Web operations are read-only and identify themselves as open-world tools.

## Non-Goals

The server does not publish, alter, or authenticate into external sites.

## Acceptance Criteria

- Search results provide a usable reference for subsequent retrieval.
- Retrieval accepts a result reference or absolute URL.

## Verification

Automated coverage verifies structured search and retrieval outputs, result limits, and failure reporting.
