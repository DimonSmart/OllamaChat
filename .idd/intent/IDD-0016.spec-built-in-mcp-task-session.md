# IDD-0016.spec-built-in-mcp-task-session

## Intent

Provide an opt-in built-in MCP server for durable task-session turns and named Markdown summaries.

## Related Specifications

Related: IDD-0007, IDD-0008.

## Behavior

Agents can create or use a bound task session, append and query ordered turns, and store or replace named Markdown summaries such as a final result or review note.

## Durable Architecture And Constraints

Session identifiers scope all stored turns and summaries.

## Non-Goals

The server does not infer task success or create an unrequested summary.

## Acceptance Criteria

- Reading turns can resume after a sequence position.
- A named summary can be replaced without corrupting other summaries.

## Verification

Automated coverage verifies session scoping, ordered retrieval, and summary replacement.
