# IDD-0005.spec-agent-templates

## Intent

Let users manage reusable agent definitions that combine instructions, optional model settings, tool access, and retrieval files.

## Related Specifications

Related: IDD-0001, IDD-0004, IDD-0006, IDD-0007.

## Behavior

Users create, search, edit, delete, and restore templates. A template can define identity, summary, instructions, optional model tuning, tool selection, MCP-server bindings, and attached retrieval files.

## Durable Architecture And Constraints

Template-specific values may defer to application defaults when omitted.

## Non-Goals

This surface does not configure server connections or author workflow source.

## Acceptance Criteria

- Required agent identity and instructions are validated.
- Attached files expose indexing state.
- Restoring shipped templates preserves custom templates.

## Verification

Automated coverage verifies template validation, model selection, tool bindings, retrieval attachments, and restoration semantics.
