# IDD-0019.spec-built-in-mcp-markdown-document

## Intent

Provide an opt-in built-in MCP server for structured reading and targeted editing of a bound Markdown document.

## Related Specifications

Related: IDD-0007, IDD-0008, IDD-0017.

## Behavior

The server exposes document context, headings, sections, searchable items, and Markdown export. It supports stateful reading cursors and applies ordered edits addressed through semantic pointers or item positions.

## Durable Architecture And Constraints

Editing is scoped to the document bound to the MCP session.

## Non-Goals

The server does not edit files outside its bound document.

## Acceptance Criteria

- A cursor can be created, resumed, inspected, and reset.
- Ordered valid operations update the bound document predictably.
- Search and export reflect the bound document.

## Verification

Automated coverage verifies cursor progression, semantic editing, document scoping, search, and export.
