# IDD-0018.spec-built-in-mcp-knowledge-book

## Intent

Provide an opt-in built-in MCP server for navigating, searching, editing, and exporting a structured knowledge book.

## Related Specifications

Related: IDD-0007, IDD-0008.

## Behavior

The server resolves its bound knowledge source, lists and retrieves sections by outline reference, inserts and updates sections, searches section content, and exports the complete book as Markdown.

## Durable Architecture And Constraints

Outline references provide a stable way to address hierarchy and edit position.

## Non-Goals

The server does not treat an unbound file as an implicit shared knowledge book.

## Acceptance Criteria

- Users can navigate root and nested outlines.
- Insert and update operations preserve a coherent outline.
- Search returns matching sections and export returns the complete book.

## Verification

Automated coverage verifies outline navigation, edits, search, and Markdown export.
