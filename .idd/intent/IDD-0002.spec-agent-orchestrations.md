# IDD-0002.spec-agent-orchestrations

## Intent

Let users load, inspect, configure, and execute saved multi-agent orchestrations.

## Related Specifications

Related: IDD-0001, IDD-0003, IDD-0005.

## Behavior

Users choose a saved workflow, inspect its participants and orchestration shape, provide declared inputs, and start a workflow chat session. The UI presents execution output and identifies the workflow session.

## Durable Architecture And Constraints

Orchestrations may express sequential, concurrent, handoff, and group-chat collaboration.

## Non-Goals

This surface does not make arbitrary workflow source code a chat message.

## Acceptance Criteria

- Users can distinguish available workflows by name, kind, and description.
- Required workflow inputs are collected before execution.
- The orchestration preview reflects its configured participants and flow.

## Verification

Automated coverage verifies each orchestration kind, input handling, and session event projection.
