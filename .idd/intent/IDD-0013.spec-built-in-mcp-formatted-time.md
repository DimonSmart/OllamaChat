# IDD-0013.spec-built-in-mcp-formatted-time

## Intent

Provide an opt-in built-in MCP server that formats current time and can ask the user for a missing time zone.

## Related Specifications

Related: IDD-0007, IDD-0008, IDD-0012.

## Behavior

The server formats current time using a supplied .NET-compatible format. When no time zone is supplied, it requests one through MCP elicitation.

## Durable Architecture And Constraints

The tool is read-only and supports interactive user input through the MCP protocol.

## Non-Goals

The server does not store user time-zone preferences.

## Acceptance Criteria

- A supplied time zone is used without elicitation.
- A missing time zone produces a clear interactive request.

## Verification

Automated coverage verifies formatting, zone resolution, and elicitation outcomes.
