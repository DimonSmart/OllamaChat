# IDD-0015.spec-built-in-mcp-user-profile

## Intent

Provide an opt-in built-in MCP server through which an agent can access and edit a user's profile preferences.

## Related Specifications

Related: IDD-0007, IDD-0008, IDD-0010.

## Behavior

The server exposes the preference document associated with its session and supports controlled updates so agents can use user-provided preferences in their work.

## Durable Architecture And Constraints

Preferences are scoped to the relevant user/session context.

## Non-Goals

The server does not expose preferences belonging to another user context.

## Acceptance Criteria

- Agents receive the resolved preference context for their session.
- Updates preserve a valid preference document.

## Verification

Automated coverage verifies parsing, scoped access, and update behavior.
