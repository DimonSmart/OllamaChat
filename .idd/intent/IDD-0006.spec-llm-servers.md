# IDD-0006.spec-llm-servers

## Intent

Manage connections to local Ollama and OpenAI-compatible model servers.

## Related Specifications

Related: IDD-0005, IDD-0009.

## Behavior

Users add, edit, test, and remove named server connections with their server type and base URL. Unavailable connections report a useful state rather than silently appearing usable.

## Durable Architecture And Constraints

The product supports local Ollama and OpenAI-compatible server protocols.

## Non-Goals

This surface does not provision models or retain provider-specific secrets in agent templates.

## Acceptance Criteria

- A connection can be identified by name, type, and base URL.
- Connection failures are visible to users.
- Deleting a connection removes it from available model discovery.

## Verification

Automated coverage verifies validation, connectivity outcomes, and model-client selection.
