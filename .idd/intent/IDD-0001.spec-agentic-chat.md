# IDD-0001.spec-agentic-chat

## Intent

Provide an interactive chat surface for running a selected agent or saved workflow.

## Related Specifications

Related: IDD-0002, IDD-0004, IDD-0005, IDD-0007, IDD-0009, IDD-0010.

## Behavior

Users select an available agent or workflow, supply required inputs, and exchange messages. Responses stream into the transcript; users can cancel an active response and copy the conversation as text, Markdown, or HTML. The transcript exposes assistant reasoning and tool activity when supplied by the runtime.

## Durable Architecture And Constraints

Conversation execution supports both direct agents and workflow-based agents.

## Non-Goals

This surface does not define how agents, workflows, servers, or tools are authored.

## Acceptance Criteria

- A valid selection can receive a user message and return streamed output.
- Cancellation leaves the conversation in a usable state.
- Users can export the displayed transcript in each offered format.

## Verification

Automated coverage verifies streaming, cancellation, transcript state, and export formatting.
