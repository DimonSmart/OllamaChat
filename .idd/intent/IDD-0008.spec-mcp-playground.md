# IDD-0008.spec-mcp-playground

## Intent

Provide a safe interactive surface for selecting an MCP server, choosing a tool, supplying its inputs, and examining its result.

## Related Specifications

Related: IDD-0007.

## Behavior

Users select an available server and function, complete fields derived from the tool input schema, invoke the function, and switch between simplified and raw result views.

## Durable Architecture And Constraints

The playground uses MCP-provided tool metadata to present invocation inputs.

## Non-Goals

This surface does not automatically attach tools to an agent.

## Acceptance Criteria

- Available functions change with the selected server.
- Required and typed inputs are represented before invocation.
- Users can inspect both concise and raw results.

## Verification

Automated coverage verifies tool-schema form creation, invocation context, and result presentation.
