# IDD-0009.spec-model-discovery

## Intent

Show models available from each configured LLM server and the capabilities relevant to agent use.

## Related Specifications

Related: IDD-0005, IDD-0006.

## Behavior

The view groups results by server and displays model name, image support, function-calling support, modification time, size, and per-server errors.

## Durable Architecture And Constraints

Model capability information is discovered from configured servers.

## Non-Goals

This surface does not download, delete, or train models.

## Acceptance Criteria

- A server with no models, a server error, and no configured servers are distinguishable states.
- Capability indicators are available for each discovered model.

## Verification

Automated coverage verifies result grouping, capability interpretation, and unavailable-server reporting.
