# IDD-0014.spec-built-in-mcp-math

## Intent

Provide an opt-in built-in MCP server for arithmetic expression evaluation.

## Related Specifications

Related: IDD-0007, IDD-0008.

## Behavior

The server evaluates expressions containing numbers, parentheses, and supported arithmetic operators, then returns the numeric result.

## Durable Architecture And Constraints

The tool is deterministic and has no external side effects.

## Non-Goals

The server does not execute general-purpose code.

## Acceptance Criteria

- Valid supported expressions return a numeric result.
- Invalid expressions return a comprehensible failure.

## Verification

Automated coverage verifies operator precedence, parentheses, and invalid-input handling.
